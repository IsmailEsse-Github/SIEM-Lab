<h1>SIEM LAB</h1>

<h2>Description</h2>
This project consists of setting up Azure Sentinel (SIEM) and connecting it to a virtual machine that is live whilst behaving as a honeypot. This lab will include the monitoring of live attacks (RDP Brute Force) from around the globe! Finally, a custom powershell script is used to locate the adversaries grographic location information and pin this on the Azure Sentinel Map.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
